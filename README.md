# Assignment-5

Purpose
The goal of this project is to use the cancer dataset to see which model gives the best performance.

Models
Three classification models were tested: Support Vector Machine (SVM), Decision Tree, and Logistic Regression.

Implementation
The dataset was divided into training and testing data using train_test_split(). Each model was trained with the training data and then used to make predictions on the test data.

Evaluation
The models were compared using accuracy, precision, recall, and a confusion matrix to see how well they classified the tumors.

Limitations
The dataset is fairly small and the models were used with default settings, so additional tuning could improve the results.
